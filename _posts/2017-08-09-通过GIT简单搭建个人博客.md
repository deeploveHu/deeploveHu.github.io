##Git简介

Git是一个开源的分布式版本控制系统，用以有效、高速的处理从小到非常大的项目版本管理。
GitHub可以托管各种git库的站点。
GitHub Pages免费的静态站点，三个特点：免费托管、自带主题、支持自制页面和Jekyll。

##为什么使用GitHub Pages

搭建简单而且免费;
支持静态脚本;
可以绑定你的域名;
DIY自由发挥，动手时间一些有意思的东西git,markdown,bootstrap,jekyll;
理想写博客环境，git+github+markdown+jekyll;
